HTML

  • Head element contains information about the webpage
  • Body element represents the visual content shown to the user
  • HTML elements are usually composed of an opening tag and a closing tag represented with angle brackets
  • The top declaration tells the browser what documenmt type it should expect ie. !DOCTYPE html to indicate HTML
  • Opening tag represents the root of our document and the attribute inside tells the browser the language for the webpage
  • Title defines the title for the webpage
  • Style defines the CSS styles associated with the webpage
  • Base defines the base URL
  • Link connects an external resource to the document
  • Meta defines metadata such as the character set, description, keywords, author, and viewport.
  • Script embeds client-side scripts in the HTML document.
  • When possible, it is important to use semantic HTML elements, because they make your HTML easier to read by other developers and the browser.
  • Content is contained between an opening tag and a closing tag. Closing tags begin with a forward slash
  • H1-h6 represent the level of heading a given text block represents.
  • Ul, ol, li represent unordered lists, ordered lists, and list items.
  • Src attribute inside the img element defines the location of the image
  • Br element creates a line of empty space, or a line break, between two blocks of content.
  • A, or anchor, element creates links to webpages. The href attribute points to the URL for the link
  • Footer appears at the bottom of the page

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • touch style.css adds CSS to your file
  • Three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS is used to style an element directly in the HTML file
  • Internal style sheet contains CSS rules for the webpage in the head section of the HTML file.
  • External CSS style sheet is used when you link the file using a link element in the HTML head element
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • * symbol is used to apply a rule to ALL the elements visible on a page

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A commits all files in the working branch
  • git commit -m "your note" associates a message with your commit
  • git pull origin main pulls in the base branch, always pull the base branch into your feature branch before pushing up any code.
  • git push origin "branch name" pushes up any changes to the code

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is a programming language that allows users to interact with websites.
  • touch script.js command is used to add JavaScript to your file
  • each line of JavaScript code ends with a semicolon.
  • console.log method is used to output a message to the web console by adding an argument.
  • Control flow is the order in which a computer executes code in a script.
  • We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • When the computer encounters an if statement in a JavaScript file, it will check to see if the condition that we provide is truthy. If it is, then it will do whatever action we ask it to do in between the curly brackets. If the condition is not true or has a value that is not truthy, then the computer will ignore whatever is in between the curly brackets and move on.
  • else if is used to have more than one condition
  • array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • In JavaScript and many other programming languages, arrays are zero-indexed and sequential.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A for loops is composed of a starting point for our loop, a condition, and a final statement that allows the array to iterate over each item
  • x++ is a shortcut way of writing x+1
  • Commenting out code makes it inactive without deleting it, just in case you need it later. Done by /* /*
  • a function is simply a set of instructions that tells the computer how to perform a certain task.
  • you must call a function by name in order to execute it.
  • A function's name is followed by parentheses
